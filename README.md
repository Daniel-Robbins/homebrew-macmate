# Homebrew Tap for MacMate

[MacMate](https://macmate.app) - The essential toolkit for headless Mac deployment.

## Installation

```bash
brew install --cask Daniel-Robbins/macmate/macmate
```

## Update

```bash
brew upgrade --cask macmate
```

## Uninstall

```bash
brew uninstall --cask macmate
```

## About MacMate

MacMate helps you deploy AI agents on headless Macs:

- **Awake Manager** - Prevent sleep without external power
- **Virtual Display** - Create virtual monitors up to 8K
- **Audio Router** - Route system audio to virtual microphone
- **System Monitor** - Real-time resource monitoring

Learn more at [macmate.app](https://macmate.app)
